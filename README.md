# The HBnB commandline Interpreter
The command line interpreter helps you to create, read, update and delete these data models listed below

-User
-City
-Amenity
-Place
-Review
-State

## How to start it
In order to start the console, you run

```
$ python ./console.py
(Hbnb) $
```
